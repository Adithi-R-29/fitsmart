# FitSmart

## Project Overview
FitSmart is a Flask-based web application developed as part of my Global Certification project.
The application is designed to provide a simple and user-friendly interface for fitness-related functionality, offering an interactive web experience using Python, HTML, and CSS.

## Features
- User-friendly web interface
- Built using Python Flask framework
- Clean HTML templates for page rendering
- Styled frontend using CSS
- Organized project structure with templates and static files

## Technologies Used
- Python
- Flask
- HTML5
- CSS3

## Installation and Setup
1. Clone the repository:
   git clone https://github.com/Adithi-R-29/fitsmart.git

2. Navigate to the project folder:
   cd fitsmart

3. Install dependencies:
   pip install -r requirements.txt

4. Run the Flask application:
   python app.py

5. Open in browser:
   http://127.0.0.1:5000/
